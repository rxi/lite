overview

lite is a lightweight text editor written mostly in lua. it aims to provide something practical, pretty, small and fast, implemented as simply as possible; easy to modify and extend, or to use without doing either.

customization

Additional functionality can be added through plugins which are available from the plugins repository; additional color themes can be found in the colors repository. The editor can be customized by making changes to the user module.

motiontrail
Adds a motion-trail to the caret
markers
Add markers to docs and jump between them quickly.
autoinsert
Automatically inserts closing brackets and quotes.
colorpreview
Underlays color values with their resultant color.
console
A console for running external commands and capturing their output.
sort
Sorts selected lines alphabetically
theme16
Theme manager with base16 themes
linter
Linters for multiple languages
workspace
Retains project's layout and open documents between sessions
